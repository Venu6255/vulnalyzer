Redis Server Management Commands:
1. Check Redis Status:
bash
sudo systemctl status redis-server
or

bash
sudo systemctl status redis
2. Stop Redis Server (temporarily):
bash
sudo systemctl stop redis-server
This stops Redis but keeps auto-start enabled for next boot

3. Start Redis Server:
bash
sudo systemctl start redis-server
4. Restart Redis Server:
bash
sudo systemctl restart redis-server
5. Check if Redis is running:
bash
redis-cli ping
Should return PONG if running

Auto-Start Management:
Disable auto-start (boot startup):
bash
sudo systemctl disable redis-server
Redis won't start automatically on boot, but you can still start/stop manually

Enable auto-start (boot startup):
bash
sudo systemctl enable redis-server
Redis will start automatically on boot

Check if auto-start is enabled:
bash
sudo systemctl is-enabled redis-server
